# yyopc

yyopc is the public home for my one-person software company.

The name is deliberately compact:

- `yyo` is my personal prefix, from my internet alias [yyovil](https://github.com/yyovil).
- `opc` stands for One Person Company.

I use this org to ship small, practical developer tools from a single-owner
workspace: local-first where possible, automation-friendly, and built around the
way one person can move quickly without pretending to be a large team.

## Projects

- [yyork](https://github.com/yyopc/yyork) - local-first orchestration for
  parallel AI coding agents. Run multiple agent sessions in isolated, durable
  workspaces and supervise them from one dashboard.

## Principles

- Keep tools close to the user.
- Prefer concrete workflows over platform ceremony.
- Make solo ownership explicit: small surface area, direct accountability, fast iteration.
